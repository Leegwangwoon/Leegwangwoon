## 🚀 Tech Adventures (주요 프로젝트)

- **TTS 모델 개발**: **VITS** 및 **SoVITS** 모델을 활용, 약 5,000개 이상의 정제된 한국어 음성 데이터셋을 학습시켜 자연스러운 고품질 음성 합성 엔진 구축
- **챗봇 개발**: 온프레미스 구축을 타겟한 로컬 모델 기반 챗봇, GPT 재미나이등 API활용한 챗봇, 에이전트와 RAG기반 및 로컬모델 파인튜닝 활용  
- **DB연동**: 서비스 호스팅을 위한 DB 연동 및 트랜잭션 관리, 데이터 임베딩 및 벡터 데이터 베이스를 활용한 유사도 검색 구현(이미지, 텍스트 등)
- **AI 서비스 호스팅 및 배포**: **LLM(Qwen, Llama)** 및 **TTS/STT** 모델을 로컬 환경에 최적화하여 온프레미스 AI 인프라 구축 및 서비스 배포
    - *주요 사례: 회의록 자동 요약, 문서 기반 RAG 챗봇, 자동 민원 응답 시스템, OCR 기반 오디오북 서비스 등*
- **영지식 증명(ZKP) & 블록체인 연구**: 이산 대수 기반의 **Leopard** 프로토콜 공동 연구 및 블록체인 데이터 모니터링 시스템 개발 (IEEE 논문 게재 및 SW 저작권 2건 등록)
